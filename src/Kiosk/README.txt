햄버거집의 메뉴 주문 시스템을 Java 콘솔 프로그램으로 구현.

사용자가 메뉴를 선택하고, 장바구니에 담고, 할인 적용 후 주문까지 진행 가능.

메뉴 출력: Burgers / Drinks / Desserts 카테고리별 출력

장바구니: 메뉴 선택 후 장바구니 추가

주문 기능: 현재 장바구니 항목 확인 및 주문

할인 적용: 국가유공자(10%), 군인(5%), 학생(3%), 일반(0%)

개발 환경

Java 11+

콘솔 기반 UI

싱글톤 패턴(Cart) 사용

메뉴 구성: MenuItem, MenuPrinter, MainMenu 등 클래스 분리
